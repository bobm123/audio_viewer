============
Installation
============


Stable release
--------------

To install Audio Viewer, run this command in your terminal:

.. code-block:: console

    $ pip install audio_viewer

This is the preferred method to install Audio Viewer, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for Audio Viewer can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/bobm123/audio_viewer

Or download the `tarball`_:

.. code-block:: console

    $ curl  -OL https://github.com/bobm123/audio_viewer/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Github repo: https://github.com/bobm123/audio_viewer
.. _tarball: https://github.com/bobm123/audio_viewer/tarball/master
